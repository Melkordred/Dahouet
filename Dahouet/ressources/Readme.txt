Guide d'utilisation de l'application :



Tout d'abord, il faut se rendre à l'adresse : https://github.com/Melkordred/Dahouet.git puis cliquer sur le dossier en bleu nommé Dahouet.

Ensuite il est nécessaire de télécharger quelques fichiers pour qu'elle puisse fonctionner correctement, les voici :

- le dossier .settings;
- le dossier img;
- le dossier lib;
- le dossier ressources;
- dans le dossier src, ne télécharger que les packages suivant : - ModelUI, - UIVoilier, - DAO;
- le reste des petits fichier tels que .classpath .gitignore, .project etc.


Une fois cela terminé, il faudra vous rendre dans le package UIVoilier pour compiler la classe Enregistrement, ce qui lancera l'application, 
et voilà vous pouvez maintenant enregistrer un nouveau voilier ainsi qu'un nouveau propriétaire.(ou plusieurs)
